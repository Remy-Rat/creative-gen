Purpose: generate the **example reference images** for the Clinical Trial Marketing Capture Brief (Section 5). These show the lab the target look for each shot. They are *style references*, not final assets — so exact packaging text isn't critical. Keep every render **face-free** (matches what the lab can actually shoot) and make the **products match the uploaded files 1:1**.

Prompt IDs below map directly to the shot IDs in the brief (A1a, A2b, etc.) so you can hyperlink each render to the matching example slot.

---

## How to use in Higgsfield

- **Model:** Soul 2 (image). Generate the still first.
- **References:** Attach the named file(s) per shot as image references and push reference strength high so the box / bowl render accurately. For shots that show both, attach all relevant files.
- **Video shots (A1, A3, C1, D1, D2):** take the approved still into **Image-to-Video** and apply the "Motion" line with a subtle / locked camera move.
- **Hand continuity (optional):** create a **Soul ID** from one hand/forearm so the same hand recurs across shots — makes the set feel like one real trial.
- **Aspect ratio:** generate **9:16 first** (social priority), then 16:9 / 4:5 where noted.
- **Negative prompt:** paste the block below into the negative field on every generation.
- **Reality check on text:** AI rarely nails tiny pack copy even with a reference. Keep the box modest in frame (or slightly soft) except in D3; for the D3 hero flat-lay, accept you may composite the real pack image over the render afterwards.
